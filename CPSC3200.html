***CPSC 3200:Object-Oriented Design***

1. Main Topics of this Class 

    This class focuses on class design and relationships.
Additionally,the class focuses on the analysis of alternative design choices, software maintainability and code reuse.

2. Concept or Technique Learned from this Class

Upon successful completion of this course, student will be able to:

  • Specify and implement contractual design.
  • Identify costs and benefits of OO relationships.
  • Distinguish between and implement OO relationships (has-a, holds-a, and is-a).
  • Define, implement, initialize and manipulate heterogeneous collections.
